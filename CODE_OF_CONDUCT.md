# Code of Conduct

## Our Standard

We want GrokSearch to be a welcoming, useful, and respectful project for maintainers, contributors, and users.

Examples of positive behavior include:

- being respectful in technical disagreement
- assuming good intent while staying specific about problems
- offering clear reproduction steps, evidence, and context
- helping others improve code, tests, and documentation

Examples of unacceptable behavior include:

- harassment or personal attacks
- insulting, dismissive, or hostile review behavior
- publishing private or security-sensitive information without permission
- derailing discussions with repeated bad-faith arguments

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, or pull requests that violate this code of conduct.

For serious concerns, contact the maintainers privately through GitHub.

## Scope

This code of conduct applies within the repository, in issues, pull requests, and project discussions.
